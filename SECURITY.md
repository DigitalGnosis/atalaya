# Security Policy

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, email **security@digitalgnosis.dev** with:

- A description of the issue
- Steps to reproduce
- The version / commit hash affected
- Any mitigations you've identified

We aim to respond within 72 hours.

## What's in scope

- Atalaya node app (Android)
- Atalaya hub (server)
- Atalaya control app
- Web dashboard
- Anything in this repository

## What's out of scope

- Vulnerabilities in upstream projects (Gemma, llama.cpp, Postgres, etc.) — please report those to upstream first
- Issues caused by unsupported configurations (rooted phones, custom ROMs, ancient Android versions outside our minimum SDK)
- Social engineering against contributors

## Disclosure

After a fix lands, we credit reporters in the release notes unless you ask us not to.
