# Phase 1 — Research

Time-boxed exploration of each Phase 1 module. **Three working days per module, hard cap.** When the cap hits, stop and commit findings to an ADR.

## Discipline

- Cite sources with repo + commit hash where possible
- Log what you tried that didn't work — that's the most valuable section
- End every note with a `## Recommendation` paragraph that informs the ADR
- If you finish in two days, don't keep researching — go start the next module

## Status legend

- `stub` — research note exists but unread / not yet started
- `in-progress` — actively being investigated; date field active
- `findings` — investigation complete, recommendation made, ADR being drafted
- `closed` — ADR committed, this note is now historical reference
