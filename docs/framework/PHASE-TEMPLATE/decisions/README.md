# Phase N — Decisions (ADRs)

Architecture Decision Records. Each captures one committed decision. ADRs are immutable once committed.

## Format

[Michael Nygard ADR template](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions). See [`TEMPLATE.md`](TEMPLATE.md).

## Status legend

- `proposed` → `accepted` → `superseded by ADR-NNNN` (or `deprecated`)
